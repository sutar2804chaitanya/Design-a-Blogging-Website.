This Blogging Website is a multi-page web project developed using HTML, CSS, and JavaScript. The project was created to demonstrate fundamental web development concepts and the implementation of various HTML elements in a practical application. It includes a Home Page, Blog Content Page, User Registration Form, Media Gallery, and Help & Support Page.

The website follows a gaming theme inspired by BGMI and showcases navigation, content presentation, multimedia integration, user interaction forms, and support information.
